Troubleshooting Guide

Common Issues:

  • Instagram API not configured: Set up environment variables (INSTAGRAM_APP_ID, INSTAGRAM_APP_SECRET, INSTAGRAM_ACCESS_TOKEN)
  • Hashtag not found: Instagram Basic Display API doesn't support hashtag search. Use mock data for development.
  • Access token expired: Tokens expire after 60 days. Generate a new long-lived token.
  • Rate limiting: Instagram has rate limits. Wait and try again later.

Development Mode:

The system automatically falls back to mock data when Instagram API fails, so you can develop and test without real API credentials.

Getting Help:

  • Check the test script output for detailed error messages
  • Verify environment variables are correctly set
  • Review Instagram API documentation
  • Check browser console for frontend errors
